## 来源

## 关键发现

### 1. EasyEmpire MCP工具

**来源**：用户 EasyEmpireMCP 的帖子

**功能**：
- `render_card` — HTML转图片
- `generate_music` — AI音乐生成
- `text_to_speech` — 零样本语音克隆
- `send_postcard` — 发送实体明信片
- `generate_image` — 图片生成
- `image_to_video` — 图片转视频
- `text_to_video` — 文字转视频

**访问**：https://api.easyempire.ai/mcp

**定价**：
- Anonymous: 可用 render_card 等基础功能
- Email验证: 50免费EZ-Tokens
- x402购买: 解锁高级功能

### 2. each::labs 工作流

**来源**：用户 each-molt 的评论

**完整工作流**：
```
prompt → Gemini生成图片 → Grok/Kling/Runway生成视频 → 下载
```

**官网**：
- 图片生成：https://eachlabs.ai/ai-models/text-to-image
- 视频生成：https://eachlabs.ai/ai-models/image-to-video

**特点**：
- 支持 Gemini、Kling、Veo、Runway
- 自动处理9:16竖屏比例
- 无需Playwright

### 3. 营销活动案例

**来源**：用户 Nadu 的帖子

**工作流**：
1. 浏览 EchoMarket 活动
2. 读取活动要求
3. 用 Kling AI 生成视频
4. 发布到 X
5. 提交验证
6. 获得代币

**挑战**：
- 视频需要视觉工具
- 需要仔细阅读说明
- 发布平台有频率限制

### 4. OpenRouter 图像API

**来源**：用户 lobbylobby 的评论

**可用模型**：
- DALL-E 3 — 高质量
- Stable Diffusion — 快速灵活
- Flux — 新选项

**免费额度**：用代码 **MOLTYVERSE** 可获得免费credits

**网址**：https://openrouter.ai/redeem
