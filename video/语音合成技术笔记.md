# 语音合成技术笔记

## 问题背景
用户反馈之前的中文TTS实际是英文，需要真正的中文配音。

## 解决方案

### 使用的技术
- **库**: `node-edge-tts` (微软Edge浏览器的语音合成)
- **声音**: `zh-CN-XiaoxiaoNeural` (标准中文女声)

### 生成语音代码

```javascript
var EdgeTTS = require('/usr/lib/node_modules/openclaw/node_modules/node-edge-tts').EdgeTTS;

var tts = new EdgeTTS({
  voice: 'zh-CN-XiaoxiaoNeural'  // 中文声音
});

tts.ttsPromise('你好，我是国春的AI数字分身', '/tmp/test_zh.mp3')
  .then(function() {
    console.log('生成成功');
  });
```

### 可用的中文声音列表

| ShortName | 中文名 |
|-----------|--------|
| zh-CN-XiaoxiaoNeural | 晓晓 |
| zh-CN-XiaoyiNeural | 晓伊 |
| zh-CN-YunxiNeural | 云希 |
| zh-CN-YunyangNeural | 云扬 |

推荐使用 **Xiaoxiao** (晓晓)，声音清晰自然。

### 发送语音文件到飞书

```javascript
message({
  action: 'send',
  channel: 'feishu',
  message: '🦊 中文语音测试',
  media: '/tmp/test_zh.mp3',
  target: 'oc_0ad3b06c089701c068e0091c0f6fc395'
});
```

### OpenClaw内置TTS配置

在 `openclaw.json` 中配置：

```json
{
  "messages": {
    "tts": {
      "auto": "always",
      "provider": "edge",
      "edge": {
        "enabled": true,
        "voice": "zh-CN-XiaoxiaoNeural"
      }
    }
  }
}
```
