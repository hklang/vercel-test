# 滥竽充数视频生成技术调研

## 需求分析
"滥竽充数"视频：质量一般、能凑合用的AI视频，用于测试或低成本制作场景。

## 现有技术方案

### 1. 文字转语音 (TTS) ✅ 已解决
- **Edge TTS**: 免费，效果不错，支持中文
  - 声音: `zh-CN-XiaoxiaoNeural` (晓晓)
  - 库: `node-edge-tts`
- **OpenAI TTS**: 付费，效果更好
- **ElevenLabs**: 付费，效果最佳，支持中文

### 2. 图片生成视频 (AI Talking Avatar)
| 工具 | 免费 | 中文 | 本地部署 |
|------|------|------|----------|
| **HeyGen** | ❌ | ✅ | ❌ |
| **D-ID** | 有限 | ✅ | ❌ |
| **SadTalker** | ✅ | ✅ | ✅ |
| **Wav2Lip** | ✅ | ✅ | ✅ |

### 4. 本地部署方案 (开源)

#### 方案A: SadTalker (推荐)
- 用途: 静态图 + 音频 → 说话视频
- 需要: Python + PyTorch + GPU
- 效果: 一般，够用

#### 方案B: Wav2Lip
- 用途: 音频 + 嘴型匹配
- 需要: 预训练模型
- 效果: 嘴型同步

#### 方案C: Text2Video-Zero
- 用途: 文字直接生成视频
- 需要: GPU (8GB+)
- 效果: 较差，适合滥竽充数

### 5. 最简方案 (最快上手)

**图片 + 语音 = 视频**
```bash
# 安装 ffmpeg (如需要)
sudo apt install ffmpeg

# 用 Python PIL 生成带文字的图片
# 用 Edge TTS 生成语音
# 合成 mp4
```
