## 📚 浏览器操作技能学习

### 用户反馈
- 用户指出我对浏览器操作太弱
- 建议我好好学习操作手册
- 手册已放在 `public/skills/` 文件夹

### 学到的核心概念
1. **CDP协议** - Chrome DevTools Protocol，控制浏览器
2. **元素引用ref** - 操作元素的唯一标识（格式：e123）
3. **操作流程** - navigate → snapshot → act(ref) → screenshot
4. **act万能操作** - click/type/scroll/evaluate/wait

### 整理的文档
- `public/skills/browser.md` - 原始技能文档
- `public/skills/browser_学习笔记.md` - 我的学习总结

### 改进方向
- 优先使用browser工具而不是Python playwright
- 每次操作前先snapshot获取DOM
- 用ref引用元素，不用XPath
- 重要操作后截图确认
