## 一、13种选号方法

| 编号 | 方法 | 说明 | 依据 | 命中率(5+) |
|:---:|:---|:---|:---|:---:|
| 1 | **遗漏值法** | 冷号回补 | 遗漏>20期优先 | 5% |
| 2 | **热号法** | 追热号 | 30期内出现≥10次 | 15% |
| 3 | **大小号法** | 均衡分布 | 小:01-10/中:11-20/大:21-30 | 13% |
| 4 | **奇偶法** | 奇偶比 | 3-4个奇数 | 30% |
| 5 | **和值法** | 和值预测 | 历史平均和值105±20 | 30% |
| 6 | **三区比** | 区间分布 | 01-10/11-20/21-30均衡 | 68% |
| 7 | **012路法** | 除3余数 | 余0/1/2各2-3个 | 30% |
| 8 | **连号法** | 连号规律 | 每期1-2组连号 | 40% |
| 9 | **同尾法** | 同尾规律 | 尾数重复概率 | 15% |
| 10 | **重号法** | 重号预测 | 与上期重复2-3个 | 78% |
| 11 | **AC值法** | AC值分析 | AC值10-12为佳 | 15% |
| 12 | **极距法** | 极距范围 | 极距18-22 | 18% |
| 13 | **周期回补法** | 遗漏回补 | 周期性遗漏回补 | 25% |

### 推荐组合
- **遗漏值 + 热号**
- **三区比 + 重号法**

### 使用流程
```
重号法 → 遗漏热号 → 调号 → 投注
```

## 三、命令行工具

### 1. 预测命令
```bash

qlc_predict.sh --auto      # 自动模式
```

### 2. 查询命令
```bash
qlc_query.sh --list        # 列出所有预测
qlc_query.sh --show 20260212  # 查看预测详情
qlc_query.sh --verify 2026018 # 验证预测结果
```

### 3. 完整参数
```bash
cd /home/lang/.openclaw/workspace/caipiao/v5_platform

python3 predict_verify.py --predict

# 验证（指定期号）
python3 predict_verify.py --verify 2026018

# 查看状态
python3 predict_verify.py --status
```

## 四、自动任务

### 1. 守护进程模式
```bash
# 启动
python3 auto_run.py --start

# 停止
python3 auto_run.py --stop

# 查看状态
python3 auto_run.py --status
```

### 2. Cron定时任务
```bash
# 每小时运行
0 * * * * /usr/bin/python3 /home/lang/.openclaw/workspace/caipiao/v5_platform/main.py

# 每2小时更新模型
0 */2 * * * /usr/bin/python3 /home/lang/.openclaw/workspace/caipiao/v6_system/run_v6.py --cron
```

### 3. Systemd服务（可选）
```ini
# /etc/systemd/system/v6-qlc.service
[Unit]
Description=QiLeCai Prediction System V6.0
After=network.target

[Service]
Type=simple
User=lang
WorkingDirectory=/home/lang/.openclaw/workspace/caipiao/v6_system
ExecStart=/usr/bin/python3 /home/lang/.openclaw/workspace/caipiao/v6_system/run_v6.py --daemon
Restart=always
RestartSec=10

[Install]
WantedBy=multi-user.target
```

## 五、验证系统

### 命中率统计
- **3+**：至少中3个
- **4+**：至少中4个
- **5+**：至少中5个（大奖）

### 验证命令
```bash
# 验证指定期号
python3 predict_verify.py --verify 2026018

# 查看统计
python3 predict_verify.py --status
```

### 统计数据存储
- `predictions/` - 预测记录
- `results/` - 验证结果
- `stats/` - 统计文件
