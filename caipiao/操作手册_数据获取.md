## 二、解析数据

### Python解析脚本

```python
import json

# 读取txt
with open('7lc_desc.txt', 'r') as f:
    lines = f.readlines()

# 解析
data = []
for line in lines:
    parts = line.strip().split()
    if len(parts) >= 10:
        data.append({
            "period": parts[0],
            "date": parts[1],
            "basic_numbers": parts[2:9],  # 7个号码
            "special_number": parts[9],
        })

# 保存JSON
with open('qlc_history_full.json', 'w', encoding='utf-8') as f:
    json.dump(data, f, ensure_ascii=False, indent=2)

print(f"解析成功: {len(data)} 条")
```

## 三、浏览器操作要点

### OpenClaw Browser工具

#### 1. 查看状态
```json
{ "action": "status" }
```

#### 2. 启动浏览器
```json
{ "action": "start" }
```

#### 3. 打开网页
```json
{
  "action": "navigate",
  "targetUrl": "https://www.17500.cn/qlc/"
}
```

#### 4. 获取DOM快照
```json
{ "action": "snapshot" }
```

#### 5. 点击元素
```json
{
  "action": "act",
  "request": {
    "kind": "click",
    "ref": "e21"
  }
}
```

#### 6. 输入文本
```json
{
  "action": "act",
  "request": {
    "kind": "type",
    "ref": "e50",
    "text": "内容"
  }
}
```

#### 7. 截图
```json
{ "action": "screenshot" }
```

## 七、问题与解决

### Gateway重启
```bash
openclaw gateway restart
# 或手动重启
kill -9 <pid>
nohup openclaw gateway > /tmp/openclaw.log 2>&1 &
```

### Browser连接超时
- 多等待几秒让服务启动
- 检查ps aux | grep openclaw-gateway
- 重试几次
