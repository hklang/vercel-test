# 🎯 七乐彩预测助手模板

## 详细版（预测助手默认显示）

```
🎯 七乐彩预测帮助

📊 上期开奖 (2026020 2026-02-25)
   02 04 12 16 21 22 24

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🔥 高命中(≥40%) - 首选

 1 📈 重号法   上期号本期可能再出
    📌 需上1期数据作为参数
    📊 参考: 02 04 12 16 21...
    ⏰ 命中率: 78%  历史: 75%

 2 📊 三区比   30个号分3区搭配
    📌 需近20期数据统计分布
    📊 参考: 1区37个 2区48个 3区55个
    ⏰ 命中率: 68%  历史: 65%

 3 🔗 连号法   相邻号码组合如14、15
    📌 需近10期数据找连号规律
    📊 参考: 连续出现号码
    ⏰ 命中率: 40%  历史: 38%

🌙 中命中(20-39%) - 辅助

 4 ⚖️ 奇偶比   单双号比例均衡3:4或4:3
    📌 需近20期数据统计奇偶
    📊 参考: 单74个 双66个
    ⏰ 命中率: 30%  历史: 32%

 5 🔢 012路   除3余数0/1/2路搭配
    📌 需近20期数据统计012路
    📊 参考: 0路47个 1路47个 2路46个
    ⏰ 命中率: 30%  历史: 28%

 6 📉 遗漏值  冷号回补遗漏>15期
    📌 需近30期数据计算遗漏值
    📊 参考: 02 04 12 16 21
    ⏰ 命中率: 25%  历史: 22%

 7 ⏰ 周期回补 固定周期号码回归
    📌 需近50期数据找周期规律
    📊 参考: 周期规律号码
    ⏰ 命中率: 25%  历史: 25%

🔵 低命中(<20%) - 谨慎

 8 📐 极距法   最大号-最小号差值
    📌 需近20期数据统计极距
    📊 参考: 极距20-25常见
    ⏰ 命中率: 18%  历史: 15%

 9 🔥 热号法   近期高频出现号码
    📌 需近10-30期数据统计热号
    📊 参考: 14(11) 28(11) 27(10) 01(10) 24(9)
    ⏰ 命中率: 15%  历史: 18%

 10 👯 同尾法  尾数相同如4、14、24
    📌 需近20期数据统计同尾
    📊 参考: 同尾号组合
    ⏰ 命中率: 15%  历史: 15%

 11 🧮 AC值法  号码组合复杂程度
    📌 需近30期数据计算AC值
    📊 参考: AC值5-8常见
    ⏰ 命中率: 15%  历史: 12%

 12 📊 大小号  小01-10/中11-20/大21-30
    📌 需近20期数据统计大小
    📊 参考: 1区37个 2区48个 3区55个
    ⏰ 命中率: 13%  历史: 15%

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

⚡ 推荐组合
 A: 重号法+三区比  → 70%
 B: 重号法+遗漏值  → 25%
 C: 奇偶比+012路   → 30%

📝 用法: 1 / 1,2 / A / B / C / 1>5
🎯 命中率: (3+)=保本 (4+)=小奖 (5+)=大奖
```

## 方法参数说明

| 方法 | 所需数据期数 | 说明 |
|------|-------------|------|
| 重号法 | 1期 | 用上期开奖号作为参数 |
| 三区比 | 20期 | 统计近20期三区分布 |
| 连号法 | 10期 | 找连号出现规律 |
| 奇偶比 | 20期 | 统计奇偶比例 |
| 012路 | 20期 | 统计012路分布 |
| 遗漏值 | 30期 | 计算各号码遗漏值 |
| 周期回补 | 50期 | 找周期性规律 |
| 极距法 | 20期 | 统计最大-最小号差值 |
| 热号法 | 10-30期 | 统计高频号码 |
| 同尾法 | 20期 | 统计同尾号码 |
| AC值法 | 30期 | 计算号码复杂度 |
| 大小号 | 20期 | 统计大小号分布 |
