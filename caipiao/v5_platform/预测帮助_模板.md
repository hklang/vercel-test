## 📖 使用指南

### 如何查看预测？

```bash
# 查看所有方法预测
python3 predict_verify.py

# 查看单个方法
python3 predict_verify.py --method 重号法
```

### 如何使用预测？

1. **单方法使用**：选择命中率高的方法，直接用其预测号码
2. **组合使用**：取多个方法预测的交集或并集
3. **参考为主**：预测仅供参考，请理性购彩
