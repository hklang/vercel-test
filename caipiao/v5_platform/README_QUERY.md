# 预测结果查询说明

## 目录结构

```
predictions/     # 预测记录
  predictions_YYYYMMDD_HHMM.json  # 按时间保存
results/         # 验证结果
  verify_YYYYMMDD.json            # 按期号保存
stats/           # 统计文件
  method_stats.json
  abcd_stats.json
```

## 使用方法

```bash
cd /workspace/caipiao/v5_platform

# 列出所有预测
python3 query.py --list

# 查询指定日期预测
python3 query.py --show 20260212

# 查询验证结果
python3 query.py --verify 2026018
```

## 示例

### 1. 列出所有预测
```
$ python3 query.py --list

历史预测记录
============================================================

  2026-02-12 16:53  [待验证]

总计: 1 条预测记录
```

### 2. 查询预测
```
$ python3 query.py --show 20260212

```

## 自动保存

每次运行 `--predict` 时自动保存预测结果到 `predictions/` 目录。  
每次运行 `--verify` 时自动保存验证结果到 `results/` 目录。
