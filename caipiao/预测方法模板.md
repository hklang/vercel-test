## 查询命令

```bash
# 查看所有预测方法说明
qlc_query.sh --help

# 查看指定方法详情
qlc_query.sh --method 重号法

# 列出所有历史预测
qlc_query.sh --list

# 查看某个预测的详情
qlc_query.sh --show 20260212_1635
```

---

## 预测命令

```bash
# 全部预测（13种方法）
qlc_predict.sh

# 单方法预测
qlc_predict.sh --method 重号法
qlc_predict.sh --method 三区比

# 组合预测
qlc_predict.sh --combo 稳扎稳打

# 自动模式（不交互）
qlc_predict.sh --auto

# 指定输出数量
qlc_predict.sh --count 5
```

# 验证最近一次预测

qlc_verify.sh --method 重号法 --period 2026018
