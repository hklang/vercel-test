# 七乐彩历史数据获取指南

## 方法1：从中彩网手动导出（推荐）

### 步骤：
1. 访问：https://www.zhcw.com/kjxx/qlc/
2. 点击 **"自定义查询"** 链接
3. 在弹出的对话框中：
   - 选择 **"按期数"** 标签
   - 输入起始期号和结束期号（例如：2024001 - 2026016）
   - 点击 **"开始查询"**
4. 等待数据加载完成后，**全选复制**数据
5. 保存为文本文件

### 期号范围参考：
- 2024-2026年: 2024001 - 2026016
- 2022-2023年: 2022001 - 2023150
- 2020-2021年: 2020001 - 2021150
- 2018-2019年: 2018001 - 2019150
- 2016-2017年: 2016001 - 2017150
- 2014-2015年: 2014001 - 2015150
- 2012-2013年: 2012001 - 2013150
- 2010-2011年: 2010001 - 2011150
- 2008-2009年: 2008001 - 2009150
- 2007年: 2007001 - 2007150

## 方法2：使用本脚本自动处理

运行：
```bash
python3 /home/lang/.openclaw/workspace/caipiao/process_manual_data.py
```

## 数据格式

保存的数据格式应为：
```
期号 日期 基本号码 + 特别号码
例：2026016 2026-02-06 02 06 10 14 18 19 23 + 11
```

## 说明

由于中彩网有反爬虫机制，自动爬取受限。
建议分批次（每年或每两年）查询并复制数据，然后合并。
