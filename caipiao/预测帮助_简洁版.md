## 📖 使用方式

```
直接输入代码：
  1      → 重号法
  1,2    → 重号+三区比
  1>5    → 先选后筛
  A      → 推荐组合A
  ?      → 预测帮助（与"预测帮助"命令相同）
  预测帮助 → 显示完整帮助（与"?"相同）
```

> 📌 提示：`?` 和 `预测帮助` 效果完全一样
