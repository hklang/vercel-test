## 🚀 3步快速开始

### 第1步：运行预测
```bash
qlc_predict.sh
```

### 第2步：查看推荐（选一个方法）
```bash
# 新手推荐（高命中率）
qlc_combo.sh 1      # 重号法（最稳）
qlc_combo.sh 2      # 三区比（均衡）
qlc_combo.sh A      # 稳扎稳打组合（重号+三区）

# 进阶选择
qlc_combo.sh 3      # 连号法（追热）
qlc_combo.sh B      # 追热防冷组合（热号+遗漏）
```

### 第3步：投注建议
- 从输出中选 **1-3组** 号码
- 每期投注 **不超过5注**
- 坚持 **3-5期** 不换方法
