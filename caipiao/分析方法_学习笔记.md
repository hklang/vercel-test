# 彩票分析方法学习笔记

## 一、关键指标

### 1. 奇偶比
- 奇数与偶数的比例
- 常见：3:4、4:3、5:2

### 2. 大小比
- 小数(01-15)与大数(16-30)的比例
- 常见：2:5、3:4、4:3

### 3. 质合比
- 质数与合数的比例
- 质数：01,02,03,05,07,11,13,17,19,23,29
- 常见：1:6、2:5、3:4

### 4. AC值（判断code值）
- 计算方法：7个号码两两相减，得到21个差值，去重后计数
- 常见范围：12-16

### 5. 和值
- 7个基本号码相加的总和
- 常见范围：81-140

### 6. 极距（跨度）
- 最大号码 - 最小号码
- 常见范围：15-28

## 二、区间分析

### 1. 三区比
- 第一区：01-10
- 第二区：11-20
- 第三区：21-30
- 示例：2:3:2

### 2. 五区比
- 第一区：01-06
- 第二区：07-12
- 第三区：13-18
- 第四区：19-24
- 第五区：25-30

### 3. 012路分析
- 0路：03,06,09,12,15,18,21,24,27,30
- 1路：01,04,07,10,13,16,19,22,25,28
- 2路：02,05,08,11,14,17,20,23,26,29
- 示例：2:2:3

## 三、关联分析

### 1. 重号
- 与上期开奖号码重复的个数
- 常见：0-3个

### 2. 连号
- 相邻的号码对数
- 常见：1-2对

### 3. 同尾
- 尾数相同的号码对数
- 常见：0-2对

# 统计历史规律
def analyze_trends(data, recent_n=100):
    recent = data[:recent_n]
    
    # 统计各指标分布
    odd_even_ratios = Counter([get_odd_even_ratio(item) for item in recent])
    sum_ranges = [get_sum(item) for item in recent]
    ac_values = [get_ac(item) for item in recent]
    
    return {
        'odd_even_distribution': odd_even_ratios.most_common(5),
        'avg_sum': sum(sum_ranges) / len(sum_ranges),
        'avg_ac': sum(ac_values) / len(ac_values),
    }

### 2. 趋势统计
```python
def trend_analysis(data, indicators=['odd_even', 'sum', 'ac'], periods=[30, 100]):
    for period in periods:
        stats = {}
        for indicator in indicators:
            values = [get_indicator(item, indicator) for item in data[:period]]
            stats[indicator] = {
                'avg': sum(values) / len(values),
                'max': max(values),
                'min': min(values),
                'distribution': Counter(values).most_common(5)
            }
        yield period, stats
```
