# 七乐彩历史数据获取说明

## 数据来源

### 1. 中彩网（官方授权）
- 网址：https://www.zhcw.com/kjxx/qlc/
- 特点：官方权威，更新及时
- 可查询：近30期、50期、100期、最多1000期

### 2. 乐彩网
- 网址：https://www.17500.cn/
- 特点：数据全面，原创分析多

### 3. 中国福利彩票官网
- 网址：https://www.cwl.gov.com/
- 特点：最权威的官方数据

## 数据格式

七乐彩每周一、三、五开奖，开奖号码由7个基本号码（01-30）和1个特别号码（01-30）组成。

## 如何手动获取

1. 访问中彩网七乐彩页面
2. 点击"近100期"或"自定义查询"
3. 将数据复制保存
