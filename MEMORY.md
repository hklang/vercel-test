## 📊 使用说明

### 命中率说明

| 标识 | 含义 |
|:---:|:---|
| **当期** | 本次预测的命中率 |
| **累计** | 所有测试的平均命中率 |
| **已测N期** | 测试了多少期 |
| **（3+）** | 命中3个及以上号码的百分比 |
| **（4+）** | 命中4个及以上号码的百分比 |
| **（5+）** | 命中5个及以上号码的百分比 |

### 累计计算

```
累计命中率 = (第1期命中率 + 第2期 + ... + 第N期) ÷ N
```
