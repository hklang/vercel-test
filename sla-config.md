# 更新SLA配置

## 更新频率
- **quiet模式**：仅阻塞 + 最终结果
- **normal模式**：里程碑更新
- **verbose模式**：每个工具调用后更新

## 里程碑定义
1. 接收任务 → 2. 理解需求 → 3. 执行搜索/操作 → 4. 整理结果 → 5. 最终回复

## 场景配置

### 快速任务（<2分钟）
- 模式：quiet
- 更新：无（完成后直接回复）

### 中等任务（2-10分钟）
- 模式：normal
- 更新：每60-90秒或里程碑时

### 长任务（>10分钟）
- 模式：normal
- 更新：每60-90秒 + 里程碑
- ETA预估："运行X（预计N分钟）"

## 阻塞处理
- 遇到任何阻塞立即询问
- 绝不自己卡住超过5分钟

## 状态查询
当用户问"还在吗？"时，应该反思是否没有主动汇报。
