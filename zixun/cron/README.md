## 创建脚本 create-weather-news-cron.sh

```bash
#!/bin/bash
# 创建每小时天气+热点新闻定时推送

openclaw cron add \
  --name "Hourly: Weather + Hot News" \
  --cron "0 * * * *" \
  --tz "Asia/Shanghai" \
  --session isolated \
  --message "请查询长春天气、今日国内要闻、国际快讯，按日报风格排版发送到飞书群，用户可回复序号查看详情" \
  --announce \
  --channel feishu \
  --to "oc_0a896a034969bc14f9fca0412a8701b4"

echo "✅ 定时任务创建成功！"
echo "📅 频率：每小时运行一次"
echo "📍 位置：Asia/Shanghai 时区"
```
