## 七乐彩彩票分析方法记录

蒋国春分享的七乐彩分析术语：

### 术语解释

| 指标 | 含义 | 筛选条件 |
|------|------|----------|
| **AC值** | 号码复杂度，反映号码差异程度 | 8、9、10、11 ✅ |
| **和值** | 7个基本号码的总和 | 81-140 ✅ |
| **首尾和** | 第一个号码 + 最后一个号码 | 21-40 ❌无符合 |
| **平均值** | 和值 ÷ 7 | 11-20 ✅ |
| **大小比** | 大数(17-30)与小数(01-16)的比例 | 2:5, 5:2, 3:4, 4:3 ✅ |
| **奇偶比** | 奇数与偶数的比例 | 2:5, 5:2, 3:4, 4:3 ✅ |
| **质合比** | 质数(02,03,05,07,11,13,17,19,23,29)与非质数比例 | 1:6, 2:5, 3:4, 4:3 ✅ |
| **三区比** | 号码在01-10、11-20、21-30三个区间的分布 | 多种组合 |

### 符号说明
- ✅ = 有符合的号码
- ❌ = 没有符合的号码
- "没有" = 该条件无解

### 七乐彩规则
- 7个基本号码：01-30
- 1个特别号码：01-30
- 每周一、三、五开奖
