# 七乐彩分析指标

## 筛选条件（2026-02-09）

| 指标 | 筛选范围 | 状态 |
|------|----------|------|
| AC值 | 8、9、10、11 | ✅ |
| 和值 | 81-140 | ✅ |
| 首尾和 | 21-40 | ❌无符合 |
| 平均值 | 11-20 | ✅ |
| 大小比 | 2:5, 5:2, 3:4, 4:3 | ✅ |
| 奇偶比 | 2:5, 5:2, 3:4, 4:3 | ✅ |
| 质合比 | 1:6, 2:5, 3:4, 4:3 | ✅ |
| 三区比 | 多种组合 | ✅ |

## 术语说明

- **AC值**：号码复杂度/差异程度
- **和值**：7个号码的总和
- **首尾和**：首号 + 尾号
- **平均值**：和值 ÷ 7
- **大小比**：大数(17-30)与小数(01-16)的比例
- **奇偶比**：奇数与偶数的比例
- **质合比**：质数与合数的比例
- **三区比**：01-10、11-20、21-30三个区间的分布

## 符号说明
- ✅ = 有符合的号码
- ❌ = 没有符合的号码
