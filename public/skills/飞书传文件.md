# 飞书传文件技能

## 何时调用
当用户提及以下内容时激活：
- "发文件"、"传文件"、"发送文件"
- "传到飞书"、"发到聊天"
- "发送本地文件"

## 可用工具

### message - 发送文件到聊天 ✅ 最常用

**操作方式**：
```json
{
  "action": "send",
  "filePath": "/path/to/file.md",
  "message": "文件说明文字",
  "target": "oc_xxx"
}
```

**示例**：
```json
{
  "action": "send",
  "filePath": "/home/lang/.openclaw/workspace/caipiao/近100期号码.md",
  "message": "📄 七乐彩近100期开奖号码",
  "target": "oc_4e000c87fddb5867d11f5d2e00a3b092"
}
```

**支持的类型**：
- Markdown (.md)
- 文本文件 (.txt)
- 任何本地文件

---

### feishu_doc - 创建飞书文档

**操作方式**：
```json
// 1. 创建文档
{ "action": "create", "title": "文档标题" }

// 2. 写入内容（替换全部）
{
  "action": "write",
  "doc_token": "文档ID",
  "content": "Markdown内容"
}

// 3. 追加内容
{
  "action": "append",
  "doc_token": "文档ID",
  "content": "追加的文本"
}

// 4. 发送链接
{ "message": "文档已创建：https://feishu.cn/docx/xxx" }
```

**返回示例**：
```json
{
  "document_id": "GdSndXYtsoVf0ix44pKca1GJnlb",
  "url": "https://feishu.cn/docx/GdSndXYtsoVf0ix44pKca1GJnlb"
}
```

**限制**：
- ⚠️ 不支持Markdown表格
- 支持纯文本、列表、代码块等

---

### feishu_drive - 云盘管理

**注意**：feishu_drive **不能上传文件**，只能：
- 列出文件夹：`{ "action": "list" }`
- 获取文件信息：`{ "action": "info", "file_token": "xxx" }`
- 创建文件夹：`{ "action": "create_folder", "name": "文件夹名" }`
- 移动文件：`{ "action": "move", "file_token": "xxx", "folder_token": "yyy" }`
- 删除文件：`{ "action": "delete", "file_token": "xxx" }`

## 注意事项

### ❌ feishu_drive 不能上传文件
feishu_drive 工具只能管理云盘文件夹，**不能上传本地文件**。

### ⚠️ feishu_doc 不支持表格
如果内容包含表格，需要：
- 方案1：转换成纯文本格式
- 方案2：用 message 直接发送文件

### ✅ message 支持所有文件类型
最简单的方式是用 message 工具 + filePath 参数。
